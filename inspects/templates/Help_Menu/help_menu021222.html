About e-Inspection

  • The e-Inspection website is accessible through any major web-browser viz. Google Chrome, Mozilla Firefox, Internet Explorer, Apple Safari etc. through the url: https://deveins.indianrailways.gov.in/
  • The website provides a convenient online platform for the inspecting officers to draft/submit their observations along with marking officers

Sign Up

First time uses:-
To sign-in, user must have to first sign-up.


This is the sign-up page. Firstly, user have to enter their valid Email Address, after entering the Email Address, Designation will be autofetched then click on Get OTP. By clicking on Get OTP user will get a pop-up message on the top of the screen with the OTP.


After that fill the Posting Status, Employee Number

After, entering the Employee Number Employee Name will be autofetched.

Hence after filling the above details then set a strong password of user account and then click on Submit button. After this user account will be created.


Sign In

Login with the user id (Login helps user to quickly and easily manage user authentication and sign-in to user website).

Users sign into a Account, provide their consent, and securely share their profile information with user platform. Customizable buttons and multiple flows are supported for user sign up and sign in.


Forgot Password

In case the user forgets his/her password then the user can easily reset it by clicking on “forgot password”.

After that, user has to fill their registered email id to receive a link to reset password.


Home Page (Dashboard)

After sign-in, user is redirected to the dashboard page providing the summary of the website. The four cards on top displaying the status of the inspection created/drafted/submitted and compliances pending/received/closed with graphics and statistics.


My Inspections

User can find several options through left sidebar:-

  • Create Notes
  • Draft Notes
  • Completed Notes
  • Pending Compliance
  • Closed Compliance
  • Received Compliance
  • Reverted Compliance


After clicking on Create Notes, new window will appear with new form. the inspecting officer can create inspection notes with the facility to mark to the relevant officers with a target date and can set priority.


Clicking on Draft Notes, new page will appear. After creating the respective above report, the officers have a choice to save it as draft or submit it.On draft, the report will be present in the draft report page which provides a facility to edit it and submit it whenever done.


Clicking on Completed Notes, new page will appear. After creating the respective above report, the officers have a choice to save it as draft or submit it. On submit, the report will be submitted and will be present in the submitted notes page with the option to view it as pdf and view the reply of the marked officers.


Clicking on Pending Compliance new page will appear. Pending Complainces has the details or list of the reports in which the marked officer has not replied to the marked observations.


Clicking on Closed Compliance new page will appear. Closed Complainces has the details or list of the reports in which the marked officer has replied to the marked observations and it has been accepted by the inspecting officer. User can also view the Inspection Note and reply on it.




Clicking on Received Compliance new page will appear. Received Complainces has the details or list of the reports in which the marked officer has replied to the marked observations. User can also view the Inspection Note and Inspections Details Done by GM/NR.




Clicking on Received Compliance new page will appear. Received Complainces has the details or list of the reports in which the marked officer has reverted the observations to the inspecting officer. User can also view the Inspection Note and Inspections Corrigendum Details Done by GM/NR.




My Compliances

User can find several options such as:-

  • Pending
  • Sent
  • Rejected
  • Reverted


Clicking on Pending new page will appear. Pending Reply, provides facility for the marked officer to view the report and reply to the inspecting officer with a choice to either forward or revert the observations. User can also view the Report and Reply on it.




Clicking on Sent new page will appear. Sent Reply, provides facility for the marked officer to see their respective reply for the marked observation. User can also view the Report and Reply on it.




Clicking on Rejected new page will appear. It informs us about the reply being rejected for the respective observation of the report generated.


Clicking on Reverted new page will appear. It informs us about the reply being reverted for the respective observation of the report generated. User can also view the Report and Remark on it.




Query

User can find several options such as:-

  • Pending
  • Sent


Clicking on Pending new page will appear. Pending Query, provides facility for the marked forward officer to view the report and reply to the officer as this contains those observation which has been forwarded. User can also view the Report and Reply on it.


After clicking on sent new page will appear. Sent Query, provides facility for the marked forward officer to see their respective reply for the marked forwarded observation. User can also view the Report and Reply on it.




RB Minutes

User can find several options such as:-

  • Create
  • Saved
  • Reply


Clicking on Create new page will appear. In Create form user can Create Minutes Of Meeting with the list of members(s) present, Date of meeting, after that the user can Add heading, Add Sub Heading and Add Decision and can also copy to different members.




After clicking on Saved new page will appear with new form. In Saved form user can view all the saved meetings.


After clicking on Reply new page will appear with new form. In Reply form user can reply to all the meetings.
