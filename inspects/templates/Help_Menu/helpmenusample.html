About e-Inspection

  • The e-Inspection website is accessible through any web-browser viz. Google Chrome, Mozilla Firefox, Internet Explorer, Apple Safari etc. through the url: https://einspection.indianrailways.gov.in/
  • The website provides a convenient online platform for the inspecting officers to draft/submit their observations along with marking officers

Sign In

This is the first page that opens when the e-Inspection website is opened. Through this form users can Login into this website using the user id and password provided by the CRIS admin. On successful login, a user dashboard with full access to their inspection record will be available to the user.


Sign Up

This feature is provided to update employee number(IPAS no.) posted on a particular designation or to change posting status, password etc related to that designation. Firstly, users have to enter their valid Email Address, if not available then user can raise a request to register email id. Once email id is entered, Designation corresponding to email entered, will be autofetched then click on Get OTP to authenticate via email OTP. On successful verification user can enter their employee number and this will change the incumbent posted on selected designation along with posting status as Primary or Dual charge. Users are required to change password during successful signup.


Forgot Password

In the event that a user forgets their password, they can easily reset it by clicking the 'Forgot Password' option available in login form. Following that, the user will need to provide their registered email address to receive an OTP to reset their password. On successful verification, user can provide a new password in a given combination and format and click on Submit button to make the changes.


Home Page (Dashboard)

After signing in, the user will be redirected to the dashboard page, which provides an overview of the website. At the top of the page, there are three cards that display the status of inspections that have been created, drafted, or submitted, as well as the status of compliances that are pending, received, or closed. Additionally, there are cards for D.O. letters, minutes of meetings, and location wise summary of inspections carried out by the officer. All data available in dashboard are interactive in nature and it will show detailed information on click tooltip is also provided for better understading.

My Inspections

This menu is given for inspecting officer where they can create, draft new inspection notes, view already issued inspection notes, new compliancestatus along with the reply received.

Following option are available in this menu accessible through side bar provided:-

  • Create Notes
  • Draft Notes
  • Completed Notes
  • Pending Compliance
  • Closed Compliance
  • Received Compliance
  • Reverted Compliance

Create Notes

Under the 'Create Notes' menu, inspecting officers can generate inspection notes and assign them to the appropriate officers with a target date, as well as set priority levels.

First of all, inspecting locations (ZR/PU/PSU/CTI) where the inspection has been conducted need to be selected from ‘Rly./Org’ field (compulsory field) then corresponding divisions/unit, if any, can be selected from ‘Div./Unit’ field (optional field). Specific Railway units like station, section, running room etc. can also be selected after selecting ‘Location Type’ (optional field) then selecting respective Railway units from ‘Location’ field. Date of inspection can be fed using double click on single date or single clicks on two dates to select range of date from ‘Insp. Date’ field (compulsory field).

After entering data pertaining to inspection, the user can set the Title of Inspection and list of officials accompanied while inspection (select designation from ‘Accompanied By’ field, if designation doesn’t exist in the dropdown then it will take the value entered by the user).

For adding inspection para, following options have been provided:

1. Add Para: can be either used to enter heading without marking to any officer or to enter inspection point along with marking to concerned officer(s).
2. Add Sub Para: to enter inspection point or sub point after adding or along with marking to concerned officer(s).
3. Initially, 'Save as Draft' button is visible and the 'Submit' button will be visible when 'Add Para' is clicked.

Under select officer field there is a filter searched where user can search on the bases of railway, divison, department and designation after that user can set the target date and priority.User can also add an image in it by selecting the reference number then select the image from your folder and upload it,and you can also seee the preview of the image. Under copy to field we send a copy of the inspection notes to the 'marked officer'.


At bottom there are three option:-

1. Save as Draft:It partially fill the data and save as draft for further editing.
2. View:user can see the preview of there inspection notes.
3. Submit:user can submit ther inspection notes.

Draft Notes

If saved as a draft, the inspection notes will be accessible on the 'Draft Report' page, where it can be edited again. On submit action further editing won't be possible and inspection note will be issued and accessible under completed notes.


Completed Notes

Under the "Completed Notes" section, user can view all list of the inspection notes issued along with the option to view inspection note as PDF and to make duplicate draft inspection note using the existing one for creating another similar inspection note. By default all inspection note are visible to user but specific inspection note can be searche between any date range or any inspection railway location or designation unit using the filter given i.e date range or rly/org wise or div/org wise.


Pending Compliance

The "Pending Compliance" section contains the details or list of inspection note which are closed and some or all of its insp. paras marked for action are pending for compliance. User can view the inspection notes in pdf form and user can view the pending reply of the marked officer. if user want a specific isnpection note can be search between any date range or any inspection railway location or designation unit using the filter given i.e date range or rly/org wise or div/org wise.


Closed Compliance

Under the "Closed Compliance" section a list of inspection notes which are closed and all of the items para marked to concerned officers have been complied and accepted by the inspecting officer(by default the compliance rec'd is accepted and inspecting officer can reject the compliance rec'd with remarks at any point of time). Users can also view the Inspection Note in PDF form and if user want a specific inspection note can be search between any date range or any inspection railway location or designation unit using the filter given i.e date range or rly/org wise or div/org wise.


Reverted Compliance

Under the "Reverted Compliance" section a list of inspection notes in which any important para marked to a particular officer is reverted back with remarks(reason for reverting) to inspection officer for marking to concerned officer. User can see the inspection note in PDF form and if if user want a specific inspection note can be search between any date range or any inspection railway location or designation unit using the filter given i.e date range or rly/org wise or div/org wise.


My Compliances

This menu is given for marked officers where they can view inspection para marked to them. sent reply or revert back to inspecting officer. provision to forward any inspection para to their subordinate officers for getting reply of any particular para if required.

Following option are available in this menu accessible through side bar provided:-

  • Pending
  • Sent
  • Rejected
  • Reverted
  • Forwarded by me

Pending

Under 'Pending' it allow the user to search for specific inspection notes between any date range or any inspection railway location or designation unit using the filter given i.e date range or rly/org wise or div/org wise. The pending reply feature allows the marked officer to view the inspection note and send compliance of an inspection para marked to them by the inspecting officer. The officer can also choose to either forward the inspection para to their subordinate officers for getting the compliance and finally sent it back to inspecting officers or revert the inspection para back to the inspecting officer with remark for reverting the correcting concerned officer. User can also view the inspection notes by click on the inspection note no. in PDF format. There are three buttons given under Action,forwarded and complied columns.



1.Action button in red color shows the number of inspection para pending for compliance and Click on the button will show a list of pending paras. The marked officer can give compliance and choose to either save para, edit the saved compliance, forward the inspection para to their subordinate officers for getting the compliance and finally sent it back to inspecting officers or revert the inspection para back to the inspecting officer with remark for reverting the correcting concerned officer. On hovering the cursor at the forwarded button,the designation of the forwarded officer can be seen.


2.Forwarded button in yellow color shows the number of inspection para forwarded. click on the button will display a list of forwarded para. The marked officer can use this feature to keep track of which replies they have already forwarded. The user can edit forwarded reply by select the rly./org, div./unit and choose to either move the received reply after moving the reply will be shown below the edit section where you can save reply or reject the reply.



3.Complied button in green color shows the number of received compliance from forwarded officer and click on the button will display a list of accepted paras.This feature allows the marked officer to keep track of forwarded paras which have been complied by the forwarded officer.


Multiple selection is given to apply an action on multiple paras simultaneously.
At bottom submit button is provided to send saved compliance of any inspection note.(before submitting it is mandatory to save the compliance first).

Sent Reply

It allows the marked officer to view their respective compliance sent marked inspection para. This feature enable the officer to search for specific inspection notes between any date range or any inspection railway location or designation unit using the filter given i.e date range or rly/org wise or div/org wise or to view previously sent compliance to inspection paras marked for the compliance. Users can also view the inspection notes by click on the inspection note number in the form of a PDF.




Rejected Reply

The Rejected feature informs the marked officer that their compliance has been rejected for a respective inspection para of any particular inspection notes. This feature enable the officer to search for specific inspection notes between any date range or any inspection railway location or designation unit using the filter given i.e date range or rly/org wise or div/org wise and also alerts the marked officer that they need to review and revise their compliance as per remarks (Rejected) send by inspecting officer.


Reverted Reply

List of Inspection notes which are marked by the inspecting officer for any inspection para to marked officer and for some reason marked officer has reverted that inspection para back to inspecting officer with reverted remarks for rectifying the marked officer to concerned officer. This feature enable the officer to search for specific inspection notes between any date range or any inspection railway location or designation unit using the filter given i.e date range or rly/org wise or div/org wise.


Forwarded by me

Under this section, officer can view list of inspection notes which me marked to the officer for any inspection para & the same have been forwarded to subordinate officer for getting the compliance of that inspection para. user can view the report in PDF form, and if user want, you can edit the forwarded compliance also.

Click on "Continue", it will display all the inspection notes. User can view the inspection report in the form of pdf and the red button indicates that the inspection is still pending or the green button indicates that designation officer has sent compliance to marked officer. for search a specific keyword by selecting "Search Insp Para" the system will show the paragraph number(s) where the word appears in all pending or closed inspection reports.

Feedback

After Clicking on Feedbacka form will appear where the user can write it's feedback.




This form includes fields such as Railway Zones, Division, Designation, Feedback Type, and Sub Category. To streamline the feedback process, the fields are dynamically activated based on the user's selection of the appropriate Category.


USER'S are also provided with the capability to check the status of their feedback. This is facilitated by a set of multiple options to choose from, including Pending, Resolved, In Progress, and Not Feasible. By selecting the desired option, officers can obtain the latest information on the progress of their feedback. Additionally, officers have the convenience of downloading comprehensive lists for each section in either Excel or PDF format.
