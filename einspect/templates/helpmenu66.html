About e-Inspection

  • The e-Inspection website is accessible through any web-browser viz. Google Chrome, Mozilla Firefox, Internet Explorer, Apple Safari etc. through the url: https://einspection.indianrailways.gov.in/
  • The website provides a convenient online platform for the inspecting officers to draft/submit their observations along with marking officers

Sign In

This is the first page that opens when the e-Inspection website is opened. Through this form users can Login into this website using the user id and password provided by the CRIS admin. On successful login, a user dashboard with full access to their inspection record will be available to the user.


Sign Up

This feature is provided to update employee number(IPAS no.) posted on a particular designation or to change posting status, password etc related to that designation. Firstly, users have to enter their valid Email Address, if not available then user can raise a request to register email id. Once email id is entered, Designation corresponding to email entered, will be autofetched then click on Get OTP to authenticate via email OTP. On successful verification user can enter their employee number and this will change the incumbent posted on selected designation along with posting status as Primary or Dual charge. Users are required to change password during successful signup.


Forgot Password

In the event that a user forgets their password, they can easily reset it by clicking the 'Forgot Password' option available in login form. Following that, the user will need to provide their registered email address to receive an OTP to reset their password. On successful verification, user can provide a new password in a given combination and format and click on Submit button to make the changes.


User Master

Admin can relinquish any officer of their railway, Post any officer in their railway, view pending request to relieve any officers presently posted in another railway and Accept/Reject request sent by another railway admin to relieve any officers presently posted in their railway.

Following option are available in this menu accessible through nav bar provided:-

  • Relinquish

  • New Joining

  • Requested

  • Pending

Designation Master

After clicking on this Designation Master Following option are available in this menu accessible through provided:-

  • All Designation Details

    • When the admin logs in with their railway ID, they will be able to see a list of details of all employees. If the admin wants to change any employee's details, they can go to the 'Action' column and modify the department, railway/division, station, reporting officer, pay level, mobile number, e-mail address (only if it is not assigned to anyone else), add remarks, and click on 'Forward Request'.

  • Add Designation

    • The super admin can directly add designations. They can select the post, which will automatically fetch the details of the department, category, pay level, and designation. Then they can select the station, reporting officer, work area, mobile number, e-mail address, and remark. In the railway/division column, when we select the railway/division, it will show whether the designation is available or not. If it is available, it will show a similar designation.

  • Requested by me

Railway Wise Report

Under the 'Railway Wise Report'admin can view railway wise summary of inspection done, closed, open, pending, overdue, and complied by the selected railway group. Reports can also be exported as Excel or PDF format or print from the option given above the table.


Division Wise Report

On clicking Division Wise Report admin can view division wise summary of inspection done, closed, open, pending, overdue, and complied by the selected divisions of any railway. Reports can also be exported as Excel or PDF format or print from the option given above the table.


Officer's Wise Report

Under the "Officer's Wise Report"admin can view officer wise summary of inspection done, closed, open, pending, overdue, and complied by the selected designation of any division or any railway. Reports can also be exported as Excel or PDF format or print from the option given above the table.
