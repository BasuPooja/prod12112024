About the Website

The website e-nireekshan which is available on any web-browser(google chrome, mozilla firefox)url: deveins.indianrailways.gov.in, It provides a platform for the inspecting officers to draft as well as submit their observations with the required marked officers following the necessary actions.

SIGN-UP

For the very first time,
To sign-in, you must first sign-up. Here is the sign-up page.Firstly you have to enter valid Email Address, certain details like designation,employee number/name will be autofetched and then on validation of the respective email. Hence After validating the OTP, set the password of user account and then click on Submit button. After this user account will be created.

SIGN-IN

After sign-up, user can sign in with the registered email id and password.Provides a facility of forgot password too, incase user forgets.

DASHBOARD

After sign-in, user is redirected to the dashboard page which provides the summary of the website e-nireekshan. The four cards on top displaying the status of the notes created/drafted/submitted and compliances pending/received/closed with graphically as well as statistically shown.

MY INSPECTION

CREATE NOTES

Here the inspecting officer can create inspection notes with the facility to mark the necessary officers with a given target date and set priority.

DRAFT NOTES

After creating the respective above report, the officers have a choice to save it as draft or submit it.On draft, the report will be present in the draft report page which provides a facility to edit it and submit it whenever done.

COMPLETED NOTES

After creating the respective above report, the officers have a choice to save it as draft or submit it.On submit, the report will be submitted and will be present in the submitted notes page with the option to view it as pdf and view the reply of the marked officers.

PENDING COMPLIANCE

Pending Complainces has the details or list of the reports in which the marked officer has not replied to the marked observations.

RECEIVED COMPLIANCE

Received Complainces has the details or list of the reports in which the marked officer has replied to the marked observations.

CLOSED COMPLIANCE

Closed Complainces has the details or list of the reports in which the marked officer has replied to the marked observations and it has been accepted by the inspecting officer.

REVERTED COMPLIANCE

Received Complainces has the details or list of the reports in which the marked officer has reverted the observations to the inspecting officer.

MY COMPLIANCES

PENDING

Pending Reply, provides facility for the marked officer to view the report and reply to the inspecting officer with a choice to either forward or revert the observations.

SENT

Sent Reply, provides facility for the marked officer to see their respective reply for the marked observation.

ACCEPTED

It informs us about the reply being accepted for the respective observation of the report generated.

REJECTED

It informs us about the reply being rejected for the respective observation of the report generated.

REVERTED

It informs us about the reply being reverted for the respective observation of the report generated.

FORWARDED COMPLIANCE

PENDING

Pending Query, provides facility for the marked forward officer to view the report and reply to the officer as this contains those observation which has been forwarded.

SENT

Sent Query, provides facility for the marked forward officer to see their respective reply for the marked forwarded observation.
